# Isolated Reference Files

Files in this folder are preserved references or disabled experiment surfaces.

- Do not delete these files just because the main Vite app does not import them.
- Do not treat these files as live game entrypoints.
- Move a file here when it is useful for comparison, recovery, or disabled work, but should not run as part of the current game path.
- Delete only files that are explicitly confirmed as deletion targets.

Current live HTML entrypoints stay at the repository root.
