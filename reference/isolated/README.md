# Isolated Reference Files

Files in this folder are preserved references, demos, or disabled visual experiments.

- Reference files and demo files are not test builds.
- Do not delete these files just because the main Vite app does not import them.
- Do not treat these files as live game entrypoints unless a README or task explicitly says so.
- Move a file here when it is useful for comparison, recovery, visual direction, or disabled work, but should not run as part of the current game path.
- Test-only files must live under `test-builds/` and use a clear name such as `테스트빌드` or `test-build`; do not use `demo` as a test-only label.
- Delete only files that are explicitly confirmed as deletion targets by exact path.

Current live HTML entrypoints stay at the repository root.
